What we measure, daily

Six numbers, one Slack channel, one screenshot at 9am. Anything more is theatre.

  • Spend — actual vs pacing, by channel
  • Free reports created — the only conversion that matters in the first 14 days
  • Cost per free report — vs the $59.92 blended target
  • Reply rate on cold-email — target ≥ 1.8% off the AmLaw-200 IP partner list
  • Trial-to-contract — measured weekly from D14 onward
  • Forecast drift — realized vs predicted CPA per channel; trigger SMT re-run if any channel drifts > 25%

The kill criteria

Cut a channel if any of:

  • CPA > 1.5× forecast for 7 consecutive days
  • Free→trial < 12% (vs 23% baseline)
  • Reply-to-meeting < 35% on outbound

Reallocate killed-channel budget to the best-performing surviving channel, capped at 50% of total.

Appendix · Method

All forecasts produced by the CampaignForge pipeline (run 2026-05-15). Components: Markov regime-switching simulator (40% weight), Google TimesFM 2.5 with XReg covariates (60% weight) executed on NVIDIA Tesla T4 via the Colab MCP bridge. Budget allocation via SMT (Z3 logical constraints + Simplex relaxation) under conservative / moderate / aggressive scenario sweep. Creative-strategy scaffolding via the ClaudeAdsBridge stage. Pipeline source: github.com/jhashemi/campaignforge. Data lineage: public docket counts (Lex Machina, PACER), Bessemer Cloud Index, three confidential expert-counsel interviews. Numbers should be re-validated at D14.
